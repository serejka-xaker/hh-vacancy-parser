## Проект: Автоматизированный парсер вакансий HH.ru с экспортом в Excel

**Август 2024 | Фриланс | Full-cycle разработка**

- Самостоятельно провёл сбор требований и коммуникацию с заказчиком, сформировал техническое решение и архитектуру приложения.
- Реализовал интеграцию с HH API с поддержкой пагинации, фильтрацией по дате публикации, региону и наличию зарплаты.
- Настроил отказоустойчивую HTTP-сессию с экспоненциальной задержкой для обработки ошибок.
- Реализовал двухэтапную загрузку данных (список вакансий → детальный запрос по ID).
- Разработал систему извлечения контактных данных и структурированных блоков («Обязанности», «Требования», «Условия») из HTML-описания.
- Внедрил конфигурируемую систему фильтрации через внешний Excel-файл, что позволило изменять бизнес-логику без изменения кода.
- Реализовал дедупликацию по ID и валидацию данных.
- Провёл ручное тестирование и отладку.
- Скомпилировал приложение в EXE-файл для передачи заказчику.

**Стек:** Python, requests, BeautifulSoup, pandas, urllib3 Retry, PyInstaller
